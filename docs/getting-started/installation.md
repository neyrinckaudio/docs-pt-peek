# Installation

## Download

Download the latest PT Peek installer from [neyrinck.com/ptpeek](https://neyrinck.com/ptpeek).

The download is a macOS installer package (`.pkg`). It is signed and notarized by Apple, so you can install it with confidence.

## Install

1. Double-click the `.pkg` file
2. Follow the installer prompts
3. PT Peek is installed to `/Applications/PT Peek.app`

## Uninstall

To remove PT Peek:

1. Quit PT Peek if it's running
2. Drag `PT Peek.app` from `/Applications` to the Trash
3. The Quick Look extension is removed automatically

## Updating

Download and run the latest installer. It will replace the previous version. Your settings and license are preserved.
