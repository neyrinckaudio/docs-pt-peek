# Keyboard Shortcuts

## Finder

| Shortcut | Action |
|----------|--------|
| ++space++ | Quick Look preview of selected `.ptx` file |

## File Management

| Shortcut | Action |
|----------|--------|
| ++cmd+o++ | Open file |
| ++cmd+shift+r++ | Reveal selected file in Finder |

## Import

| Shortcut | Action |
|----------|--------|
| ++cmd+shift+i++ | Import selected items to Pro Tools |

## Navigation

| Shortcut | Action |
|----------|--------|
| ++up++ | Select previous item |
| ++down++ | Select next item |
| ++shift+up++ | Extend selection up |
| ++shift+down++ | Extend selection down |
| Click | Select item |
| ++cmd++ + Click | Toggle item in selection |
| ++shift++ + Click | Extend selection to item |

## Playback

| Shortcut | Action |
|----------|--------|
| ++space++ | Play/pause selected audio file or clip (in session browser) |
